Controlling a Quadrotor

Jake Dluhy

A Math 198: Hypergraphics Project

Abstract

In this instructional webpage, we will explore each of the aspects that go into controlling a quadrotor. By starting with the basics of kinematics, progressing through dynamics, and into the world of state space modeling, we will build up the basic knowledge that goes into quadrotor design. In the end, we will see everything put together into a game that will test your control engineering skills!

1. Kinematics
The kinematics of motion. Changing coordinate frames from one to the next, including translations and rotations. After learning the lesson, there will be an interactive animation to move the quadrotor, visualizing it from the reference frame of the environment, as well as the reference frame of the quadrotor itself.
2. Dynamics
The dynamics of a quadrotor. How forces and torques are applied on the body, and how the quadrotor responds appropriately. At the end of the lesson, try to apply forces to each quadrotor blade and attempt to manually control the quadrotor. It might be harder than you think!
3. Manual Control
Controlling a quadrotor by positioning a desired position, and manually choosing gains. At the end, see how quadrotor controls work, setting a specified destination objective for the quadrotor to fly to.
4. Optimal Control
A better method than manually choosing gains is deciding how to minimize the cost for the quadrotor. See how these costs are derived, and how changing the weights applied to each one can speed up or slow down a quadrotor. Be carefull not to pick too aggressively, in case this causes the quadrotor to fail!
5. The Target Game
The final chapter is just the animation for the target game. This will test your skills as a control engineer. Five targets are placed within the arena. Choose the costs you think will make the quadrotor perform best, and see if you can set a new personal record!
